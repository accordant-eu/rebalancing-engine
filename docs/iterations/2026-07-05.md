### Decisions
1. **Utility-First Styling:** Removed global CSS files in favor of standardizing the entire application on Tailwind CSS v4, improving consistency and reducing style leakage.
2. **Iconography:** Replaced placeholder emojis and manual SVGs with `lucide-react`, standardizing the visual language with a clean, vector-based icon set (ADR-0057).
3. **Typography & Aesthetics:** Replaced default browser fonts with the 'Inter' sans-serif stack. Applied a "premium SaaS" design language incorporating subtle glassmorphism, depth through gradients, rounded components, and purposeful whitespace.
4. **Information Architecture Correction:** Moved the `AssetUniverseTab` out of the Tenant Admin layout to the Superadmin layout exclusively, reflecting its global scope and authorization rules (`requireSuperadmin`).

### Open Questions / Known Issues
- Currently, "Action Inbox" routes to a placeholder within the `AdvisorLayout`. The actual inbox logic is scheduled for a future tranche.
- Form components in the admin tabs currently rely on standard HTML form behaviors. More complex forms could benefit from `react-hook-form` and client-side schema validation in future updates.
